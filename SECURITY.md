# Security Policy

## Supported Versions

Currently, only the latest release (`main` branch) is supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| < 1.3.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly. 

Instead, open a private issue or contact the maintainer directly. We will try to address the issue within 72 hours.
